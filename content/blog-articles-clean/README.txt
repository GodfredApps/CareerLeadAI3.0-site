FORMATTING GUIDELINES:

Headings:
## Main Section (will become H2 in Sanity)
### Subsection (will become H3 in Sanity)

Lists:
Use simple dashes or numbers:
- Item one
- Item two

1. First item
2. Second item

No special formatting needed - Sanity will handle it.

Emphasis:
Write naturally. You can add bold/italic in Sanity's editor after pasting.

Links:
Write naturally. Add hyperlinks in Sanity's editor.

Examples/Quotes:
Use plain text without quote marks or code blocks.
