## üìù How to Add to Sanity Studio

### Step 1: Access Studio
Navigate to: https://careerlead-cms.sanity.studio/

### Step 2: Create New Blog Post
1. Click "Blog Post" in the sidebar
2. Click the "+" button to create a new post

### Step 3: Fill in Fields
For each article, copy from the summary above:

**Required Fields:**
- Title
- Slug (must be unique, lowercase with dashes)
- Excerpt (short description for preview)
- Category (dropdown: create categories if needed)
- Author (create Author document first)
- Reading Time (number in minutes)
- Featured (toggle to Yes for these 5 articles)
- Published At (set to current date/time)

**Content:**
- Copy the entire markdown content from the `.md` file
- Paste into the "Content" field (it supports rich text/markdown)

**Optional but Recommended:**
- Target Audience (select from dropdown or add new)
- SEO Meta Title (same as title or optimized version)
- SEO Meta Description (use the excerpt)
- Career Tags (link to relevant career paths once created)
- Skill Tags (link to the 5 skills you already added)

### Step 4: Publish
1. Review the preview
2. Click "Publish" button (top right)
